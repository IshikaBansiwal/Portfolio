View Project

This project implements a registration system where users can register by providing their details, and the information is stored in an Excel spreadsheet. The system allows users to sign up by entering personal data such as name, email, contact number, and address. The data is then automatically added to an Excel file, acting as a simple database. This project serves as a basic introduction to data handling and automating data storage.
